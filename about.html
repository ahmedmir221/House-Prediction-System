Technologies and Tools Used

  • Backend: Python with Flask - This lightweight web framework powers the backend of our application, making it easy to build and maintain.
  • Machine Learning: Scikit-Learn - This powerful library allows us to create a robust predictive model, helping users get reliable house price estimates based on their inputs.
  • Model Training: Random Forest Regressor - We chose this method for its ability to handle complex datasets and produce accurate predictions by combining the results of multiple decision trees.
  • Data Handling: Pandas - This essential tool simplifies data manipulation and analysis, enabling us to preprocess the information effectively.
  • Frontend: HTML, CSS, JavaScript - These foundational web technologies shape the user interface, ensuring a seamless and engaging experience for the users.
  • Deployment: Localhost - For now, our application runs locally
